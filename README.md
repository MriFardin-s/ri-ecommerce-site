# QurbaniHat - Digital Haat Management System


**QurbaniHat** is a modern, full-stack web application designed to simplify the process of buying and selling animals . This platform provides a beautiful experience for users to browse, filter, and view detailed information about animals listed for the upcoming Qurbani season.



## Project Purpose
The primary goal of QurbaniHat is to provide a **secure, user-friendly, and efficient digital marketplace** for cattle and other animals trading. By leveraging modern web technologies, the project ensures data integrity and a premium user experience through real-time data fetching and robust security protocols.


## Key Features

### Advanced Security & Auth
*   **Secure Authentication:** Integrated with **Better Auth**, supporting both traditional Email/Password and **Google Social Login** for a friction-less experience.
*   **Protected Routes:** Critical pages like `/animals` and `/my-profile` are guarded by **Next.js Middleware** and server-side session checks.

### Performance & UX
*   **Dynamic Data Management:** Real-time data fetching with dynamic sorting capabilities (Price: Low to High / High to Low) using URL search parameters.
*   **Optimized UI/UX:** Crafted with **HeroUI** (formerly NextUI) and **Tailwind CSS** for a fully responsive and aesthetic interface.
*   **Fluid Animations:** Enhanced with **Animate.css** for engaging component transitions and entry effects.
*   **Seamless Feedback:** Utilizes **Suspense boundaries** and custom **loading skeletons** to ensure a smooth perceived performance during data loads.
